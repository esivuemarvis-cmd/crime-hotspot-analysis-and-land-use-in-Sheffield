# Sheffield Crime Hotspot Analysis and Land Use

This GIS project analyses crime hotspots in Sheffield and explores their relationship with land use patterns.

## Project Description

This project investigates spatial crime patterns in Sheffield using GIS analysis.  
Hotspot mapping was used to identify high crime concentration areas and compare them with surrounding land use types.

## Tools Used

- ArcGis
- Spatial analysis
- Kernel density / hotspot analysis
- Land use classification
